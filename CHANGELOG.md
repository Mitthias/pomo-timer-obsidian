# Changelog

All notable changes to Pomo Timer will be documented here.

## [1.0.0] — 2026-04-25

### Added
- Work / Short Break / Long Break Pomodoro cycle
- SVG ring progress indicator
- Click-to-edit time on the clock face (type `MM:SS` or plain minutes)
- Quick-set preset chips with add / remove support
- Last-set time remembered across relaunches
- Collapsible stats: Today, This week, 4-month activity heatmap
- Desktop notifications (Obsidian Notice + system)
- Status bar indicator with live countdown
- Theme-aware colors (follows Obsidian accent)
- Commands: Play/Pause, Skip, Reset, Open panel
- Settings: all durations, long break interval, session-end behaviour, notifications
